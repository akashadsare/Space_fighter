ðŸ”§ Challenges & Solutions

The development journey wasn't without obstacles. Here are some key challenges I faced:

Memory Management

Working in C meant handling memory manually. I implemented a custom memory pool to reduce fragmentation and improve performance during intense gameplay sequences.

Performance Optimization

To maintain 60 FPS even with numerous entities on screen, I developed spatial partitioning for collision detection and implemented object pooling for frequently created/destroyed entities like projectiles.

Cross-Platform Compatibility

Using SDL2 helped create a cross-platform foundation, but I still needed to write platform-specific code for file handling and some input methods.
